# Laravel Task Manager

Простой таск-менеджер (список задач) на Laravel с поддержкой AJAX (удаление и переключение статуса без перезагрузки страницы) и кэшированием списка задач.

## Стек технологий

- PHP 8.2+
- Laravel 11
- MySQL (или PostgreSQL)
- Bootstrap 5 (интерфейс)
- JavaScript (fetch, AJAX)
- Laravel Cache (кэширование списка задач)

## Основные возможности

- Просмотр списка задач, отсортированных по дате выполнения (ближайшие первыми)
- Добавление новой задачи через модальное окно
- Редактирование задачи (название, описание, дата) на отдельной странице
- Удаление задачи без перезагрузки страницы (AJAX)
- Отметка о выполнении (переключение статуса) без перезагрузки страницы (AJAX) с визуальным выделением строки
- Кэширование списка задач (сброс кэша при любом изменении данных)

## Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/ваш-логин/task-manager.git
cd task-manager
2. Установка зависимостей
bash
composer install
3. Настройка окружения
Скопируйте файл .env.example в .env:

bash
cp .env.example .env
Отредактируйте .env, указав параметры подключения к базе данных:

env
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=task_manager
DB_USERNAME=root
DB_PASSWORD=
Также установите APP_URL (например, http://localhost:8000).

4. Генерация ключа приложения
bash
php artisan key:generate
5. Запуск миграций и сидеров (опционально)
bash
php artisan migrate
php artisan db:seed --class=TaskSeeder   # создаст несколько тестовых задач
6. Запуск встроенного сервера
bash
php artisan serve
